# Arduino-based-ESP32-Game-Controller


This project turns an ESP32 into a Bluetooth gamepad using physical buttons.

## Features
- 4 Buttons:
  - Accelerate
  - Brake
  - Left
  - Right
- BLE Gamepad support
- Works with PC / Android

## Pin Configuration

| Function    | GPIO |
|------------|------|
| Accelerate | 25   |
| Brake      | 26   |
| Left       | 27   |
| Right      | 14   |

## Library Used
- BleGamepad
